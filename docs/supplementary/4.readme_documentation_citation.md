## Quick Start
```python
from cellclusterflow import Pipeline

pipeline = Pipeline()
adata = pipeline.load_data("data/pbmc3k.h5ad")
adata = pipeline.run_qc(adata)
adata = pipeline.normalize(adata)
adata = pipeline.cluster(adata)
pipeline.plot_umap(adata)
```
