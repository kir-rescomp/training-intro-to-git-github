# GitHub Authentication 🔐

GitHub requires authentication to push code or access private repositories. **Passwords no longer work** - you must use SSH keys or tokens.

---

!!! circle-info "Choose Your Method"

    === "🔑 SSH Keys (Recommended)"

        **Best for:** Daily development work

        - ✅ Set up once, works forever
        - ✅ No passwords needed
        - ✅ Industry standard

        **Setup time:** ~10 minutes

        [:octicons-arrow-right-24: SSH Keys Guide](2.setting_up_ssh_keys.md)

    === "🎫 Personal Access Tokens"

        **Best for:** HTTPS access, automation, CI/CD
      
        - ✅ Works through firewalls
        - ✅ Fine-grained permissions
        - ✅ Can cache credentials
      
        **Setup time:** ~5 minutes
      
        [:octicons-arrow-right-24: Access Tokens Guide](3.tokens.md)
      


---

## Quick Comparison

| Feature         | SSH Keys          | Access Tokens    |
| --------------- | ----------------- | ---------------- |
| **Convenience** | ⭐⭐⭐⭐⭐             | ⭐⭐⭐⭐             |
| **Setup**       | One-time          | Can expire       |
| **Firewalls**   | Sometimes blocked | Usually works    |
| **Best for**    | Personal use      | HTTPS/automation |

---

## Not Sure Which to Use?

- **New to Git?** → Start with [SSH Keys](ssh-keys.md)
- **Behind firewall?** → Use [Access Tokens](access-tokens.md)
- **Want both?** → Set up both! (SSH for yourself, tokens for scripts)
