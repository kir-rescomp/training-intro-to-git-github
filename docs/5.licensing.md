# <h1 align="center"> Licensing</h1>

!!! clipboard-list "Objectives"

    - Explain why adding licensing information to a repository is important.
    - Choose a proper license.
    - Explain differences in licensing and social expectations.



As an open source project, Software Carpentry relies on volunteers to create our lessons and includes a file named LICENSE or LICENSE.txt file in all public lesson repositories. This file is used to specify that all materials are freely available under the Creative Commons Attribution license. Without a file that clearly states under which license any public source code, manuscript or other creative works is being made available, the default copyright laws apply.
